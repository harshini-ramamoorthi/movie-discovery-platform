# 🎬 CineFlow – Movie Discovery Platform

CineFlow is a modern movie discovery web application built using React and Vite that allows users to explore trending movies, search titles, view movie details, and manage favorites through a responsive Netflix-inspired UI.

## 🚀 Live Demo

https://cineflowmoviediscovery.netlify.app

## ✨ Features

* 🔍 Search movies dynamically
* 🎥 Browse trending and popular movies
* ❤️ Add and remove favorites
* 📄 Detailed movie information pages
* ⚡ Fast and responsive UI powered by Vite
* 🌙 Modern Netflix-inspired dark interface
* 📱 Fully responsive across devices
* ⏳ Skeleton loading states for improved UX
* 🔄 Real-time API data fetching

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/harshini-ramamoorthi/movie-discovery-platform.git
```

Move into project:

```bash
cd movie-discovery-platform
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_API_KEY=your_api_key_here
```

Run locally:

```bash
npm run dev
```

Build production:

```bash
npm run build
```

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
